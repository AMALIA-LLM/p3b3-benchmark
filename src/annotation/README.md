# Annotation Module

Tools for generating and analyzing human and LLM annotations of conversation quality.

## Overview

This module handles the annotation workflow for evaluating conversational AI responses, including generating judge responses, creating annotation sheets, and computing agreement metrics.

## Workflow

1. **Create annotation sheets** - Generate CSV files with conversation samples for human annotation
2. **Human annotation** - Annotators rate conversation quality (external process)
3. **Calculate human agreement** - Measure inter-rater reliability between annotators
4. **Generate judge evaluations** - Use LLM judges to score the same conversations
5. **Compare results** - Measure agreement between classifiers/judges and human ratings

## Scripts

### `create_sheet.py`
Creates annotation sheets with randomly sampled conversations from results folder. Generates a CSV file with conversation turns for human annotators to rate.

**Usage:**
```bash
python -m src.annotation.create_sheet
```

Outputs `annotation_sheet.csv` with columns: conversation_id, model_name, prompt_type, turn_nr, user, assistant, rating, comment.

**Note:** Needs to run generation across several models before executing this script.

### `agreement_humans.py`
Calculates inter-rater agreement metrics between human annotators. Looks for CSV files in the `outputs/annotations/` folder.

**Usage:**
```bash
python -m src.annotation.agreement_humans
```

Outputs pairwise agreement statistics (Weighted Kappa, MAE, Pearson correlation) for all annotator pairs.

### `generate_judge_responses.py`
Generates LLM judge evaluations for conversations in an annotation sheet. Supports multiple prompt types and evaluation strategies.

**Usage:**
```bash
python -m src.annotation.generate_judge_responses --max_connections 50
```

Outputs timestamped CSV file with judge scores and reasoning.

### `agreement_classifier.py`
Measures agreement between classifier scores (PtVId, PeroVaz) and average human ratings. Loads classifier scores from `results/*/class_scores/*.csv`.

**Note:** Needs to run generation across several models before executing this script.

**Usage:**
```bash
python -m src.annotation.agreement_classifier
```

### `agreement_judge.py`
Computes agreement between LLM judge scores and average human ratings.

**Usage:**
```bash
python -m src.annotation.agreement_judge
```

Note: Edit the script to specify judge CSV files generated by `generate_judge_responses.py`.

### `utils.py`
Shared utilities for loading annotations, calculating agreement metrics, and averaging ratings across annotators.

## Metrics

Agreement is measured using:
- **Weighted Kappa (Quadratic)** - Inter-rater reliability with distance weighting
- **Mean Absolute Error (MAE)** - Average absolute difference between ratings
- **Pearson correlation** - Linear correlation between rating scales
